About

Air connect API

A simple, open project exploring airport data. We combine public datasets into a fast, searchable experience powered.

Our mission

Make aviation data approachable for everyone — developers, hobbyists, and students. Air connect API brings together airport details, regions, countries, comments, and sample routes in one place.

What we do

We ingest public CSV and JSON sources, normalize them, and expose a clean API and a lightweight UI. You can filter, page, and browse airports or explore them on an interactive map.

Data sources

  • OurAirports CSV datasets (airports, countries, regions, comments).
  • OpenFlights for sample airline/route information.
  • airline-route-data project (compiled airline route samples).

Our values

Simplicity
A small, focused codebase that’s easy to read, run, and extend.
Openness
Open source licensing, open data sources, and transparent architecture.
Practicality
Rational defaults, fast responses, and useful endpoints with filters and pagination.
